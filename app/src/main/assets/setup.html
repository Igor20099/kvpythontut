Установка Kivy

Установка на Windows

Сначало нужно обновить pip, wheel и virtualenv:
python -m pip install --upgrade pip wheel setuptools virtualenv 
Устанавливаем зависимости:
python -m pip install docutils pygments pipiwin32 kivy_deps.sdl2==0.1.22 kivy_deps.glew==0.1.12
python -m pip install kivy_deps.gstreamer==0.1.17
Установка Kivy:
python -m pip install kivy==1.11.1
Также можно установить примеры Kivy:
python -m pip install kivy_examples==1.11.1
Примеры устанавливаются в каталог share в корневой каталог, где установлен Python
Чтобы запустить пример введите:
python {путь к корневой папке python}\share\kivy-examples\demo\showcase\main.py
Более подробно по установки на Windows можно посмотреть на официальном сайте kivy.org

Установка на OS X

Установка Kivy
$ python -m pip install kivy
Для воспроизведения мультимедии с Kivy нужно установить ffpyplayer
$ python -m pip install ffpyplayer
Обязательно нужно установить переменную KIVY_VIDEO = "ffpyplayer" перед запуском приложения
Более подробно по установки на OS X можно посмотреть на официальном сайте kivy.org

Установка на Linux

Устанавливаем обновления pip:
$ python -m pip install --upgrade --user pip setuptools virtualenv
Устанавливаем Kivy:
$ python -m pip install kivy
Также можно установить примеры Kivy:
$ python -m pip install kivy_examples
Для воспроизведения мультимедии с Kivy нужно установить ffpyplayer:
$ python -m pip install ffpyplayer
Обязательно нужно установить переменную KIVY_VIDEO = ffpyplayer перед запуском приложения
Более подробно по установки на Linux можно посмотреть на официальном сайте kivy.org
