Атрибут id

Для того чтобы использовать виджет из разметки Kv Design language, этому виджету можно присвоить id. Для этого напишите атрибут id:название у виджета. Далее создайте переменную и присвойте ему название id виджета.
Рассмотрим на примере. Создадим приложение где у нас будет макет BoxLayout с виджетами Text Input и Label.
